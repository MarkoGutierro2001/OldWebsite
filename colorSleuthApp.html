Color Sleuth App

Description: The purpose of this project was to learn problem solving skills. I was required to either fix or add code based on the given situation.
The result is a game where the player must choose the block that is a slightly different color from the others.

K - What I knew | L - What I learned | W - What I want to Learn

K - I knew that debugging was important.
L - I learned multiple methods of efficiently debugging my programs.
W - I want to learn what to do if I can never find out what is wrong with my program.
